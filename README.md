# 🌧️ Rainfall Prediction & Canal Risk System

## 📌 Overview
This project predicts rainfall for villages in Andhra Pradesh and identifies high-risk areas near canals.

## 🚀 Features
- Rainfall prediction using Machine Learning
- Canal proximity analysis using GIS
- Risk classification (Low, Medium, High)
- Streamlit dashboard for visualization

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- GeoPandas
- Streamlit

## 📊 Output
- Full village prediction table
- Canal-based high-risk alert table

## 🎯 Use Case
Helps in agriculture planning and flood risk management.
